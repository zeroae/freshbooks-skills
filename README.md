# FreshBooks Skills

Guided workflows for the FreshBooks MCP server. These skills turn raw API tools into conversational business workflows for freelancers and small business owners.

## Prerequisites

- FreshBooks MCP server connected and authenticated

## Skills

| Skill | Description |
|-------|-------------|
| `invoice-client` | Create and send an invoice using client/item names |
| `log-and-bill` | Convert tracked time entries into an invoice |
| `business-pulse` | Get a plain-English financial health summary |
| `month-end-close` | Guided month-end bookkeeping checklist |
| `expense-capture` | Quickly log an expense with smart categorization |
| `client-onboard` | Set up a new client with optional project and estimate |

## Autonomy Model

- **Read-only actions** (reports, lookups): Run automatically
- **Write actions** (create, send, delete): Always confirm first
